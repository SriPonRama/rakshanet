# RakshaNet Frontend

React + Vite frontend for RakshaNet disaster management platform.

## Tech Stack

- React 19 + Vite
- Tailwind CSS
- React Router
- Leaflet.js for maps
- Framer Motion for animations
- Axios for API calls

## Setup

```bash
npm install
```

## Environment

Create `.env`:
```env
VITE_API_URL=http://localhost:5000/api
```

## Run

```bash
npm run dev
```

Runs on http://localhost:5173

## Build

```bash
npm run build
```

Output in `dist/` folder.

## Features

- 5 role-based dashboards
- Animated landing page
- Leaflet maps with heatmaps
- JWT authentication
- Responsive design
- Professional UI
